# Playwright Automation Learning

This repository contains my learning journey in Playwright automation testing.

## Tools & Technologies
- Playwright
- JavaScript
- Node.js
- Visual Studio Code

## Project Description

In this project, I am creating automated test scripts for different web applications to practice various automation testing concepts such as:

- Login automation
- Form interactions
- Dropdown handling
- Input validations
- UI element verification
- Navigation testing

Different demo websites are used to practice automation scenarios.

## How to Run the Tests

1. Install dependencies
npm install

2. Run tests
npx playwright test
